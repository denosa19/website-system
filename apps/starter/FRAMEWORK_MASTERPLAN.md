# Internet Firma Framework – Masterplan v1

## Ziel

Das Internet Firma Framework ist die technische Grundlage für wiederverwendbare Kundenwebsites.

Es soll ermöglichen, professionelle Webseiten schneller, einheitlicher und skalierbar zu erstellen.
