# Website System

## Zweck

Dieses Repository enthält das wiederverwendbare Website-System der Internet Firma.

Ziel ist es, neue Kundenwebseiten möglichst schnell, einheitlich und in hoher Qualität zu erstellen.

---

## Struktur

00_docs → Dokumentation

01_starter → Grundgerüst neuer Webseiten

02_components → Wiederverwendbare Komponenten

03_sections → Fertige Webseitenbereiche

04_layouts → Seitenlayouts

05_templates → Komplette Website-Vorlagen

06_sanity → CMS-Struktur

07_deployment → Deployment und Hosting

08_assets → Logos, Icons, Bilder und Dateien

09_prompts → Prompts für AI Studio und KI-Entwicklung
